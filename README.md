# phd-thesis
Reference copy of my UC Berkeley Ph.D. dissertation (2015).

The focus of my PhD was methods for scaling statistical machine learning algorithms to big data. My dissertation is titled, "Scalable Ensemble Learning and Computationally Efficient Variance Estimation" and was awarded the 2015 Erich L. Lehmann Citation by the UC Berkeley Department of Statistics (granted to an outstanding Ph.D. dissertation in theoretical statistics).
